# SusMode: Sustainable Browser Extension

A Chrome extension designed to promote eco-friendly browsing habits and help users reduce their digital carbon footprint.

## Installation

### Prerequisites
- Google Chrome browser (version 88 or later)
- The extension source code cloned or downloaded to your local machine

### Installation Steps

1. **Clone or download the repository**
   ```bash
   git clone <repository-url>
   cd susmode
   ```

2. **Open Chrome Extensions Page**
   - Open Google Chrome
   - Go to `chrome://extensions/` (or: Menu → More Tools → Extensions)

3. **Enable Developer Mode**
   - In the top-right corner of the Extensions page, toggle **Developer mode** to ON

4. **Load the Extension**
   - Click the **Load unpacked** button
   - Navigate to the SusMode extension directory
   - Select the folder and click **Open**

5. **Verify Installation**
   - The SusMode extension should now appear in your Extensions list
   - Click the SusMode icon in your Chrome toolbar to open the extension popup
   - You should see the SusMode welcome screen with sustainable browsing features

### Uninstalling the Extension

1. Go to `chrome://extensions/`
2. Find "SusMode: Sustainable Browser Extension"
3. Click the **Remove** button
4. Confirm the removal

## Features

- Monitor browsing habits
- Track energy consumption
- Get eco-friendly suggestions
- Support sustainable web practices

## Development

To make changes to the extension:
1. Edit the HTML, CSS, or JavaScript files as needed
2. Go to `chrome://extensions/`
3. Click the **refresh icon** on the SusMode extension card to reload changes
4. The popup will reflect your updates on next click

## Troubleshooting

- **Extension not showing up?** Make sure Developer mode is enabled
- **Changes not appearing?** Click the refresh button on the extension card
- **Permissions errors?** Check that the manifest.json file is valid JSON
